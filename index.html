BD 서버 가이드

골드 경제, 상점, 부동산, 메이드, 펫, 레벨 시스템을 한 번에 확인하는 공개 플레이어 가이드입니다.

Java 주소
Java 포트 25565
Bedrock 주소
Bedrock 포트 19132
서버 접속 시 자바 리소스팩은 자동 다운로드됩니다. 돈, 부동산 증서, 메이드 증서, 펫 상자 같은 커스텀 아이템은 리소스팩을 허용해야 정상 표시됩니다.

Economy

골드와 은행

서버 화폐는 골드(G)입니다. 기본 소지금은 250G이며, 은행에서 실물 지폐로 교환할 수 있습니다.

전자 골드

상점 구매, 판매, 세금, 메이드 월급, 펫 상자 구매에 사용됩니다.

/money

은행

100G, 1,000G, 10,000G 지폐로 인출하거나 다시 입금할 수 있습니다.

/bank

소지금 표시

인벤토리를 열면 잠깐 보유 골드가 표시됩니다. 숫자는 보기 쉽게 정리되어 표시됩니다.

Markets

상점과 시세 변동

NPC를 클릭해서 상점을 엽니다. 일반 플레이어는 직접 상점 소환 명령어를 쓰지 않습니다.

건축

건축 상점

블록, 장식, 작업용 아이템을 거래합니다. 건축 상점 구매가는 낮춰져 있고 판매가는 별도 기준을 따릅니다.

광물

광물 상점

광석과 광물류를 거래합니다. 판매 가격은 마크 시간 7일마다 0.8배에서 1.2배 사이로 변동됩니다.

농산물

농산물 상점

작물과 음식류를 거래합니다. 판매 가격은 마크 시간 7일마다 0.5배에서 1.5배 사이로 변동됩니다.

전리품

전리품 상점

몹 전리품과 희귀 드롭을 거래합니다. 판매 가격은 0.5배에서 2배까지 변동되며 높은 가격대는 더 낮은 확률로 등장합니다.

마법

마법 상점

경험치 병, 물약 재료, 일부 마법 관련 아이템을 거래합니다. 가격은 마크 시간 7일마다 변동됩니다.

상점에서 거래할 때 구매, 판매, 일괄 판매 기록은 서버 콘솔에 남습니다.

Land

부동산

부동산 증서를 사용하면 현재 청크를 내 땅으로 등록합니다. 내 땅에서는 다른 플레이어의 설치와 파괴가 막히고, 상자는 보호됩니다.

항목 내용
첫 토지 첫 청크 구매 가격은 10,000G입니다.
추가 구매 구매할 때마다 다음 증서 가격이 10,000G씩 올라갑니다.
토지세 1개는 무료, 2개부터 초과 토지당 마크 시간 7일마다 50G입니다.
소유권 해제 확인창을 거치며 500G 수수료가 필요합니다. 해제하면 부동산 증서를 돌려받습니다.
권한 부여 믿을 수 있는 플레이어에게 토지 권한을 줄 수 있습니다.
/land info /land list /land claim /land unclaim /land trust <닉네임> /land untrust <닉네임> /land map

Maid

메이드

메이드는 내 땅에 설치해 농사, 아이템 줍기, 동물 먹이주기를 돕는 보조 NPC입니다.

고용

부동산 관리인에게 메이드 증서를 구매한 뒤, 내 땅에서 땅을 우클릭하면 해당 메이드가 소환됩니다. 메이드는 한 명만 보유할 수 있으며 해고 후 다시 고용할 수 있습니다.

가격 20,000G

월급

마크 시간 7일마다 200G가 지출됩니다. 잔액이 부족하면 보유 골드가 마이너스가 될 수 있습니다.

7일마다 200G

농사

현실 시간 2시간마다 연결된 내 소유 토지 전체에서 성숙한 작물을 전부 수확합니다. 밀, 당근, 감자, 비트, 네더 사마귀, 호박, 수박을 처리합니다.

/maid chest

상자 지정

상자를 바라보고 명령어를 사용하면 보관 상자가 지정됩니다. 지정 직후에도 한 번 작업을 확인합니다.

/maid info
메이드 작업 범위는 메이드가 설치된 토지와 상하좌우로 붙어 있는 같은 소유자의 토지입니다. 떨어져 있는 다른 내 땅에는 적용되지 않습니다.

Pets

펫

로미에게 펫 상자를 구매하면 20% 확률로 아기 펫을 얻습니다. 펫은 주인을 따라다니며 보관함과 호감도 시스템을 가집니다.

펫 상자

가격은 8,000G입니다. 펫이 나오지 않으면 일반 보상이 지급됩니다.

펫 확률 20%

Levels

전투, 생활, 채굴 레벨

서버 활동으로 별도 스킬 경험치를 얻고, 레벨이 오르면 보너스가 적용됩니다.

전투 레벨

몬스터 처치로 경험치를 얻습니다. 레벨 10마다 최대 체력, 5마다 공격력과 방어력, 20마다 이동속도 보너스가 붙습니다.

/combatlevel

생활 레벨

작물 수확, 동물 먹이주기, 낚시로 경험치를 얻습니다. 낚시는 5레벨마다 두 배 보상 확률이 5%씩 오릅니다.

/lifelevel

채굴 레벨

광석 채굴로 경험치를 얻습니다. 5레벨마다 채굴 속도가 조금 오르고, 10레벨마다 추가 광물 확률이 증가합니다.

/mininglevel

Consumables

귀환서와 도핑 물약

로미는 펫 상자와 귀환서를 판매합니다. 도핑 물약은 일정 기간 성장 효율을 올립니다.

귀환서

사용하면 서버 스폰 위치로 이동합니다. 가격은 1,000G입니다.

1,000G

경험치 도핑

전투, 생활, 채굴 경험치 획득량을 마크 시간 3일 동안 50% 올립니다.

2,000G

쓰다듬기

펫을 우클릭하면 보관함이 열리고, 하루에 한 번 호감도도 같이 오릅니다.

Troubleshooting

자주 생기는 문제

커스텀 아이템이 종이, 금 조각, 스폰 알처럼 보여요.

리소스팩이 적용되지 않은 상태입니다. 서버 접속 시 리소스팩 다운로드를 허용하고, 이미 접속 중이면 재접속하세요.

메이드가 작물을 안 캐요.

보관 상자가 지정되어 있는지, 작물 수확 토글이 켜져 있는지, 완전히 자란 작물인지, 메이드 땅과 붙어 있는 내 토지인지 확인하세요.

메이드가 24개만 캐던 문제는 어떻게 됐나요?

현재 설정은 작물 수확 제한 0으로, 0은 무제한입니다. 현실 시간 2시간마다 연결된 내 토지의 성숙 작물을 전부 처리합니다.

다른 사람이 내 땅에서 상자를 열 수 있나요?

일반 플레이어 토지에서는 소유자와 권한을 받은 사람만 상자를 사용할 수 있습니다. 국가 토지는 공용 시설로 쓰기 위해 일부 상호작용이 열려 있습니다.

리소스팩은 친구도 자동으로 받나요?

자바는 서버 리소스팩 설정으로 자동 다운로드됩니다. 베드락은 Geyser 리소스팩을 통해 접속 시 내려받습니다.
